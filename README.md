# File Integrity Monitor (FIM)

**File Integrity Monitor (FIM)** — настольное приложение для мониторинга целостности файловой системы в Linux.  
Программа позволяет отслеживать изменения файлов, выявлять модификации, удаления, ошибки доступа и нарушения целостности данных с использованием криптографических методов.

Проект разработан на языке **C++** с использованием **Qt 6** и встроенной базы данных **SQLite**.

---

## Возможности

- добавление каталогов для мониторинга;
- рекурсивное сканирование файловой системы;
- вычисление хешей файлов (SHA-256);
- обнаружение изменений, новых и удалённых файлов;
- контроль метаданных (права доступа, владелец, время модификации, inode);
- защита записей базы данных с помощью **HMAC (SHA-256)**;
- ведение истории сканирований;
- фоновый режим работы;
- уведомления через системный трей;
- графический интерфейс с таблицами, фильтрацией и поиском.

---

## Используемые технологии

- **Язык программирования:** C++
- **GUI:** Qt 6 (Qt Widgets)
- **База данных:** SQLite
- **Хеширование:** SHA-256
- **Подпись данных:** HMAC (SHA-256)
- **Платформа:** Linux
- **Формат распространения:** AppImage

## Архитектура проекта

Проект имеет модульную архитектуру и разделён на три логические подсистемы:

### Ядро (`core`)
Реализует основную логику мониторинга целостности:
- обход файловой системы;
- сбор метаданных;
- вычисление хешей;
- сравнение текущего и предыдущего состояния файлов;
- формирование результатов сканирования.

### Хранилище данных (`storage`)
Отвечает за работу с базой данных SQLite:
- хранение текущего состояния файлов;
- запись истории изменений;
- управление транзакциями;
- проверка целостности записей с использованием HMAC.

### Графический интерфейс (`gui`)
Обеспечивает взаимодействие с пользователем:
- управление каталогами мониторинга;
- запуск и остановка сканирования;
- отображение результатов и истории;
- фоновый режим и уведомления;
- выполнение сканирования в отдельном потоке.
